# Roshan Narawade - Full Stack Developer Portfolio

A modern, responsive personal portfolio website showcasing my work as a Full Stack Web Developer from Pune, India.

## 🎯 Overview

This portfolio demonstrates my expertise in full-stack development with a focus on:
- **Frontend**: React.js, Next.js, HTML5, CSS3, Tailwind CSS, Bootstrap
- **Backend**: Node.js, Express.js, REST APIs, JWT Authentication
- **Database**: MySQL, MongoDB
- **Cloud & DevOps**: AWS (EC2, RDS, S3, Route 53), Vercel, Google Cloud
- **Tools**: Git, GitHub, Clerk, Cloudinary, Resend

## 🛠️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/itsmeroshan12/react-tailwind-personal-portfolio.git

# Navigate to project directory
cd react-tailwind-personal-portfolio-main

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📦 Dependencies

- react: ^18.2.0
- lucide-react: ^latest
- @emailjs/browser: ^latest
- tailwindcss: ^latest

## 🌐 Environment Variables

If using EmailJS for contact form:

```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```
